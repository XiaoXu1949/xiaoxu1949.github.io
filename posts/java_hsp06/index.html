StringBuffer 类常见方法

//增
s.append(',');// "hello,"
s.append("张三丰");//"hello,张三丰"
s.append("赵敏").append(100).append(true).append(10.5);//"hello,张三丰赵敏 100true10.5"

//删
/*
* 删除索引为>=start && <end 处的字符
* 解读: 删除 11~14 的字符 [11, 14)
*/

System.out.println(s);//"hello,张三丰赵敏 true10.5"
//改
//老韩解读，使用 周芷若 替换 索引 9-11 的字符 [9,11)
s.replace(9, 11, "周芷若");
System.out.println(s);//"hello,张三丰周芷若 true10.5"
//查找指定的子串在字符串第一次出现的索引，如果找不到返回-1
int indexOf = s.indexOf("张三丰");

//插
//老韩解读，在索引为 9 的位置插入 "赵敏",原来索引为 9 的内容自动后移
s.insert(9, "赵敏");

5.Math 类

//看看 Math 常用的方法(静态方法)
//1.abs 绝对值
int abs = Math.abs(-9);
System.out.println(abs);//9
//2.pow 求幂
double pow = Math.pow(2, 4);//2 的 4 次方
System.out.println(pow);//16
//3.ceil 向上取整,返回>=该参数的最小整数(转成 double);
double ceil = Math.ceil(3.9);
System.out.println(ceil);//4.0
//4.floor 向下取整，返回<=该参数的最大整数(转成 double)
double floor = Math.floor(4.001);
System.out.println(floor);//4.0
//5.round 四舍五入 Math.floor(该参数+0.5)
long round = Math.round(5.51);
System.out.println(round);//6
//6.sqrt 求开方
double sqrt = Math.sqrt(9.0);
System.out.println(sqrt);//3.0

//7.random 求随机数
// random 返回的是 0 <= x < 1 之间的一个随机小数
// 思考：请写出获取 a-b 之间的一个随机整数,a,b 均为整数 ，比如 a = 2, b=7
// 即返回一个数 x 2 <= x <= 7
// 老韩解读 Math.random() * (b-a) 返回的就是 0 <= 数 <= b-a
// (1) (int)(a) <= x <= (int)(a + Math.random() * (b-a +1) )
// (2) 使用具体的数给小伙伴介绍 a = 2 b = 7
// (int)(a + Math.random() * (b-a +1) ) = (int)( 2 + Math.random()*6)
// Math.random()*6 返回的是 0 <= x < 6 小数
// 2 + Math.random()*6 返回的就是 2<= x < 8 小数
// (int)(2 + Math.random()*6) = 2 <= x <= 7
// (3) 公式就是 (int)(a + Math.random() * (b-a +1) )
for(int i = 0; i < 100; i++) {
System.out.println((int)(2 + Math.random() * (7 - 2 + 1)));
}

//max , min 返回最大值和最小值
int min = Math.min(1, 9);
int max = Math.max(45, 90);
System.out.println("min=" + min);
System.out.println("max=" + max);

常含敬畏之心

学习里的听和做

知行合一具体到学习中，最近有体会。听先闻道者讲，花一分气力时间；自己来动手实践，至少应该花两分气力。通俗点就是上课花一个小时听讲，课下最少应该花两个小时来记忆、琢磨、回顾等。否则，这个信息传递过程中，损失的细节以及自己的感悟丢失太多，会导致一知半解，这比没有灌输知识更加糟糕。再有兴趣是最好的老师，兴趣又和每个人的天性以及成长经历有关。

别贪不属于自己的东西

20216.20

刚一个不认识的实验室学姐回实验室收拾东西，忽然尴尬的发现自己贪了她的书。事情是这样的，研三的一个学姐读博去了，留下一个位置，以为没人坐了，桌上有几本书，三本linux的书，还有三本行草。本着肥水不流外人田，便没有经过物主人的同意，擅自拿走了。

今天物主人回来了，还在找这些书，虽然她不知道在我这里，但自己觉得惭愧，恨不得找个地缝钻进去，就是贪小便宜。事情没有败露，却给自己提个醒，见微知著。不是自己的东西不要妄想占有，不要贪小便宜，只拿属于自己的东西就好，也不要贪得无厌，多了不一定好。

回头看，拿了哪些书，又有什么用呢！都没怎么看过，不看何必要占有哪些书呢？竟然占有了，就要物尽其用。现代人就是过度消费，消费这些物质，消费感情。再豪华的房子，住着不安心；再奢侈的轿车，开着不踏实；房间里堆再多的书，不看；手里拿着再多的钱，没有真感情；这又有什么用呢！看似富足，实则贫乏。

做到光明磊落，心如止水，坦坦荡荡。这些更多的是心，你的心也即映射了这个世界。要做到我心光明，我心光明了，任何事情都不要害怕，上天会帮助你的！

2021.5.21向他汇报

这次汇报时，心情极度不好。

这次汇报，自己没有什么成果、进展，又为小论文的事情着急。老师又一个劲的催着做一些自己很难做出来得东西，因此导致心情极度沮丧，晚上给老妈通了电话，心情才有所平复。

当时写的日记：

今天又和他汇报，没啥用，感觉他也是在完成任务。为什么呢？就是没有实质性的帮助，再加上他说的话，给的命令，完全就没啥帮助。我希望有张科备那种实打实的帮助，或者郭淼那种精神上的帮助都可以。然而，我再怎么严格要求自己，他都无感。或许他自身难保，自己的事情都处理不了呢！不管了！按照自己的进度来就好了。

按照自己的进度来，事情要做，但是要按照我自己的节奏来，不管你说什么，除非有实质性的帮助，不然都按照我自己的方式来做。目标不会变了，就是这个追踪器加分割目标。数据集有了，最大的麻烦是啃下代码，代码怎么实现的，以及实现的原理。然后才是怎么修改，实验。实验是一小部分。按照实际情况，应该是先写，把该写的东西写好，在写的基础上，做实验，就是把写的东西弄好看，再给一点依据。

做吧，按照自己的方法，按照自己的节奏来！

总结：首先自己要努力，内源动力最总要。其次，对于别人的要求 量力而行，要结合自己的实际来做事，有自己的节奏最重要。最后，不管怎么样，都是要自己对自己负责滴，父母再厉害也只能帮你某些方面；父母再不厉害，也能在某些方面帮助你。言而总之，尽人事，知天命。每个人的能力都是有限的，尽力去做了，享受了过程，对于结果，不可控，秉承着顺其自然，船到桥头自然直就好。
